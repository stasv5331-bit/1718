# Консольное приложение с тремя заданиями

## Описание
Программа реализует три алгоритмических задания с консольным меню.

## Задания
1. Сумма двух массивов с сортировкой
2. Сумма/разность больших чисел (массивов цифр)
3. Поворот матрицы на 90°

## Требования
- Python 3.7+

## Запуск
```bash
python main.py
